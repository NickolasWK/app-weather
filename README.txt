ВИДЖЕТЫ С ПОГОДОЙ

Данный сайт позволяет создавать виджеты и просматривать погоду (температура, скорость ветра) по заданным координатам.

Соответствие требованиям из ТЗ:
+ Написано без библиотек и фреймворков
+ валидация данных
+ Асинхронность
+++ Код понятен
++ Дизайн (показывается сила ветра, иконки погоды меняются)
+++ Возможность добавить несколько виджетов
+++++++ Интегрированная карта Яндекса (+ несколько виджетов)
+++++ Деплой приложения (возможность посмотреть через GitHub Pages)
